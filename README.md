# Civic-guide
CivicGuide is a high-fidelity React app built with Vite, designed to simplify democracy. It features a real-time Google Civic API lookup for polling places and representatives, a Claude-powered nonpartisan AI assistant, and an interactive election timeline. All wrapped in a premium, dark-mode "Academic" aesthetic.

🏛️ CivicGuide
Your Nonpartisan Companion for the Democratic Process
CivicGuide is a high-fidelity React application designed to make civic engagement approachable, transparent, and engaging. By combining real-time government data with AI-driven education, CivicGuide helps users navigate the complexities of registration, voting logistics, and representative democracy.

✨ Features
📍 Live Civic Lookup: Real-time integration with the Google Civic Information API. Users can search any US address to find their polling locations, upcoming election dates, and ballot contests.

🤖 AI Civic Assistant: A specialized, nonpartisan educational chat interface (powered by Claude/Anthropic) that explains complex topics like the Electoral College, mail-in voting, and audit processes.

📅 Interactive Election Timeline: A curated journey through the voting process, from "Months Before" (Registration) to "After Voting" (Certification).

⚖️ Nonpartisan by Design: Built-in system prompts ensure neutrality, focusing entirely on the process of democracy rather than candidates or parties.

🎨 Premium UI/UX: A bespoke "Academic-Dark" aesthetic featuring:

Responsive layouts and fluid animations.

Glassmorphism and radial gradient backgrounds.

Tailored typography using the Cormorant Garamond and Crimson Text families.

🛠️ Technical Stack
Framework: React (Vite)

API Integrations:

Google Civic Information API

Anthropic Claude API

Styling: CSS-in-JS & CSS Keyframe Animations

Security: Environment variable management for API protection
